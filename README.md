# Aedo

## 장례부고 서비스

- 부고를 모바일로 간편하게 작성하여 전송하는 서비스
- 구글 플레이스토어에서 다운로드 가능

---

## 진행

### 기간

- 2022.01.17 ~ 2022.04.26

### 기여도

- 백엔드 부분을 혼자 맡아 개발하였다.

---

## 사용 기술

- Node.js / Express
- Mongoose
- Mongodb Atlas
- NaverCloudPlatform
- gitlab

---

## 주요 구현 내용

- Rest API 설계
- JWT를 이용한 자동 로그인 기능
- 부고작성, 1:1문의, 공지사항 등 다양한 게시판 만들기
- Sens API를 활용한 문자 인증

---

## 아쉬운 점

- NoSQL DBMS를 사용했지만, 데이터간의 관계 설정을 피할 수 없었다.
- 편의를 위해 일시적으로 시행한 규칙들이 Restful하지 못했다. (수정하기에 너무 많은 비용이 소모됨)
